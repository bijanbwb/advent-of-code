## Scripts

Here are example commands for running things like formatting, a local server,
and the test suites:

## Elm

```sh
$ npm run format
$ npm run start
$ npm run test
$ npm run watch
```

## Elixir

```sh
$ mix format
$ iex -S mix
$ mix test
$ mix test.watch
```
